Bulma Installation Instructions

Bulma is very easy to install since the framework is technically just one large css stylesheet.

  1. Place the following code into your html file inside the <head> tag.
    2. <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bulma@0.9.3/css/bulma.min.css">
  2. You're all done!

Alternatively, you may also download the bulma css stylesheet and manually create your own link instead of using the one one.

  1. Navigate to https://bulma.io/ and click the download button in the top right corner.
  2. Open the provided zip folder and navigate to /bulma/css to extract the bulma.css file.
  3. Place the following code into your html file inside the head tag (Don't forget to fill in the path to where you placed bulma.css).
    4. <link rel="stylesheet" href="(path to bulma.css)/bulma.css">
  4. You're all done!

Django Installation Instructions

Installing Django is very easy, but requires a few dependencies.

  1. Ensure that you have python downloaded. You can get python here.
  2. (Optionally) Look into setting up a virtual environment where you can work with Django if you do not wish to install Django systemwide.
  3. Enter the following command into your machine/virtual environment command line.
    4. Windows: py -m pip install Django
    UNIX / Linux: python -m pip install Django
  4. You're all done!
