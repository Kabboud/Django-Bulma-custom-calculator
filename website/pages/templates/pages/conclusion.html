Bulma / Django Review

Bulma Review

  • Installation Difficulty: Bulma was extremely easy to install since all I really had to do was link the stylesheet and the instructions were provided easily on the website, so all it took was a quick copy and paste.
  • Page Creation Difficulty: Given that I already had knowledge of bootstrap, it was as if I had always used Bulma. They are basically identical in usage so if you know one, you are already capable of using the other. All I lacked was knowledge about the specific classes of bulma, but this information was easy to find in Bulma's excellent documentation.
  • Learning Curve: Due to my knowledge of bootstrap, I was ready to use Bulma out of the box right away. It was extremely easy to learn and use.

Django Review

  • Installation Difficulty: I already had python set up and ready to go so this was very easy to install with a quick pip install command.
  • Setup Difficulty: Django is unlike the other frameworks we used in class in a lot of ways. It took a bit more research to find the information I needed to properly get everything set up and ready to use relative to other frameworks we've used in class, but it wasn't too bad overall.
  • Performance Expectations: Django was capable of doing everything I needed it to do and I was able to meet all my goals for the functionality of the website and calculator in particular.
  • Learning Curve: The learning curve was a bit steeper than other frameworks I've used with regard to learning how Django reads directory organization, but once I managed to get everything set up to work the way I intended, all that really remained was coding and since this was entirely done in python which I am very familiar with, it was very easy to get the functionality I needed working.

Overall

  • Would I do things differently?: No, overall, I was pleased with the pace of how quickly I managed to get everything working and I greatly enjoyed learning Django in particular.
  • Did I regret my choices?: No, not at all, I managed to get everything done in a reasonable amount of time and without too much frustration.
